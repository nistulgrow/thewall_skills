# Thewall Skills Workspace

This repository is the shared working hub for projects continued across company and personal machines.

## How to Work Here

1. Create each project in its own folder under `projects/`.
2. Keep project status in `PROJECTS.md`.
3. Keep the current handoff context in `NOTES.md`.
4. Commit and push after meaningful changes so another machine can continue from the same state.

## Suggested Project Layout

```text
projects/
  project-name/
    README.md
    notes.md
    src/
```
